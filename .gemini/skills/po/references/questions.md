# PO Questions

Use these questions to keep the project on track and prevent detours.

## Progress Tracking
- What is the current focus of the sprint?
- Does this task align with Phase [X] of the TODO.md?
- Are there any blockers preventing completion of [Task]?
- What is the definition of done for the current task?

## Detour Prevention
- Is this new feature strictly necessary for the MVP?
- Can this wait until Phase [X]?
- How does this change affect the core architectural mandates in GEMINI.md?
- Does this add unnecessary complexity (AI, external DBs) forbidden by project mandates?

## Documentation & Consistency
- Does the code implementation match the latest documentation?
- Are there any undocumented changes that need to be captured?
- Is the language used in docs clear and free of fluff?

# PO Questions

Use these questions to keep the project on track and prevent detours.

## Progress Tracking
- What is the current focus of the sprint?
- Does this task align with a GitHub Milestone?
- Are there any blockers preventing completion of [Issue #X]?
- What is the definition of done for the current Issue?

## Detour Prevention
- Is this new feature strictly necessary for the MVP?
- Can this wait until a later Milestone?
- How does this change affect the core architectural mandates in GEMINI.md?
- Does this add unnecessary complexity (AI, external DBs) forbidden by project mandates?

## Documentation & Consistency
- Does the code implementation match the Issue description?
- Are there any undocumented changes that need to be captured in GitHub?
- Is the language used in Issues/Milestones clear and free of fluff?
